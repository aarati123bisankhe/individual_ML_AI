# Frontend

This folder is the starting structure for the Smart Grocery Navigation and Scan-to-Pay mobile app.

Suggested app flow:

1. login / register
2. home dashboard
3. product search
4. product location and navigation
5. barcode / QR scan
6. digital cart
7. scan-to-pay checkout
8. receipt and order history
9. manager dashboard

Main feature modules:

- `auth`
- `home`
- `product_search`
- `store_navigation`
- `scan_to_pay`
- `cart`
- `payment`
- `recommendations`
- `manager_dashboard`
